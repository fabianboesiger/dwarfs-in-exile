About

Dwarfs in Exile has a longstanding tradition dating back to 2015 where the first edition was created. Back then, it was a simple offline Java application with similar mechanics as the current version. You could send your dwarfs to gather resources, craft items and build up your base. This original version is still downloadable here.

The current version is a complete rewrite of the original game using modern web technologies. The game also has more items, multiplayer mechanics such as quests and your dwarfs work for you even when you are offline.
